# 文件对比工具总结

* 最新版本：`v1.0`
* 更新时间：`20210914`

## 简介

总结各种文件内容比较工具。包括专业的非常好用的Beyond Comapre，以及常用的基本够用的VSCode的文件比较，以及其他一些文件比较工具，比如Notepad++的Compare插件的文件比较，WinMerge，diffuse，KDiff3，Meld等。
